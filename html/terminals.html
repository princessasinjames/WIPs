Terminals

Terminals are a pretty interesting part of coding. Traditionally, we learn to run code using a terminal. Though there is a concept called a virtual environment where you can write and run code. On a mc, you could type and enter "-m venv venv", then "source venv/bin/activate".

Example binary:

  1. 0000 = 0
  2. 0001 = 1
  3. 0010 = 2
  4. 0011 = 3
  5. 0100 = 4
  6. 0101 = 5
  7. 0110 = 6
  8. 0111 = 7

Binary

Binary is the language that computers use to understand inputs.
