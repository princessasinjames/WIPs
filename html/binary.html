Binary

Binary is a language of 1s and 0s. Binary is typically used to represent numbers. These numbers start at 0 and increase by adding a 1 and moving it forward if there was already a 0 there. I have placed a more visual example below. The example I included is a 4 bit number, with each character being a bit. There are of course larger binary numbers. If the example was of only 1s (i.e 1111), then its value would be 15. This is because limit of a 4 bit number is 16. The limit of an n bit binary number is 2^n, because the number starts at 0.

Example binary:

  1. 0000 = 0
  2. 0001 = 1
  3. 0010 = 2
  4. 0011 = 3
  5. 0100 = 4
  6. 0101 = 5
  7. 0110 = 6
  8. 0111 = 7
