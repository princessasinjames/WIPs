HTML and CSS

HTML is a language commonly used for web design.This webpage was made with HTML. HTML is an arguably easier coding language to learn since it mainly uses letters or words to format text, instead of manipulating numbers and variables.

Interesting HTML operations:

  1. a
  2. video
  3. img
  4. source
  5. style

Binary

I will now describe the functions above. "a" is used with href to create a link to the given address. "video" is used with "source" to add videos to a webpage. Using "img" for video does not work, although using a gif does. "img" is used with src to add images to a webpage. "style" is like interior CSS code (CSS that does not have to be placed into another file and linked to the HTML file). "style" is one of my favorite HTML operations. "style" can allow you to change a wide variety of text formats. For example: color, background-color, font-size, text-align, and margin-left.
